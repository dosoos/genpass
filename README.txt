# What Is sjmm
 generation a random password

# How To Install?
```sh
$ pip install sjmm
```

# How To Use?
import sjmm
sjmm.gen() or sjmm.gen(10)

## example:
```sh
$ sjmm.gen(10, type=1)
$ 8214465316
```

```sh
$ sjmm.gen(10, type=2)
$ wsqsteyqml
```


```sh
$ sjmm.gen(10, type=3)
$ 8bvgotpiex
```


```sh
$ sjmm.gen(10, type=4)
$ CQPOADQCZH
```


```sh
$ sjmm.gen(10, type=5)
$ 9X5215N66X
```


```sh
$ sjmm.gen(10, type=8)
$ |[=}-'*=[]
```


```sh
$ sjmm.gen(10, type=9)
$ |*6"|8-`$>
```
